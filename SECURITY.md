# Security Policy

## Supported Versions

Only the latest released major version of RINslider is actively maintained.
Older versions may not receive security updates.

Users are encouraged to always use a fixed, up-to-date version when consuming the CDN files.

---

## Reporting a Vulnerability

If you discover a potential security vulnerability, please report it **privately**.

**Do not open public GitHub issues for security-related reports.**

### Contact
- Email: rinws.slider@gmail.com
  *(or alternatively via GitHub private message)*

Please include:
- A clear description of the issue
- Steps to reproduce (if applicable)
- A minimal proof of concept (if possible)

You will receive a response as soon as reasonably possible.

---

## Scope

This repository contains **prebuilt, static CDN artifacts** only.

Security considerations mainly apply to:
- client-side execution in the browser
- supply-chain integrity of published files

Build tooling, source code and development dependencies are maintained outside of this repository.
